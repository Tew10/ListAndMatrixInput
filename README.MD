# ListAndMatrixInput

## Description
Java console application that allows users to enter data in lists or matrices. Users can register names, numbers, or heights, view the stored values, and work with both single-dimensional lists and two-dimensional arrays using loops.

## Features
- Enter data in **lists** or **tables (matrices)**
- Register **names**, **numbers**, or **heights**
- Display all stored values
- Works with **ArrayLists** and **2D arrays**
- Simple console-based interaction

## How to Use
1. Run the program in a Java IDE or console.
2. Choose whether to work with a **list** or a **matrix**.
3. Enter the size of the list or the number of rows and columns for the matrix.
4. Select the type of data to register: names, numbers, or heights.
5. Input the values as prompted.
6. View the stored values displayed on the console.
7. To exit, select the exit option when prompted.

## Requirements
- Java 8 or higher
- Any IDE that supports Java (Eclipse, IntelliJ, NetBeans) or command line
